## 📦 컴포넌트 목록

### BASICS
- Button — 다양한 변형을 지원하는 버튼
- Badge — 상태, 카테고리 표시 뱃지
- Spinner — 로딩 스피너
- Skeleton — 플레이스홀더

### OVERLAY
- Modal — 다이얼로그
- Toast — 알림 컴포넌트
- Tooltip — 호버 정보 표시
- Notification — 알림 드롭다운

### Navigation
- Tab / Breadcrumb / Pagination
- Sidebar Nav / Stepper

### Data Display
- Accordion / Carousel / Progress Bar
- Card / Image Zoom

### FORM
- Form Input / Toggle Switch / Dropdown
- Search Autocomplete / Chip Filter
- Star Rating / File Upload

### Interactive
- Counter / Countdown Timer
- Clipboard Copy / Scroll Progress
