> 로컬스토리지 이용하여 다크모드의 모드 이름을 기억해서 다시 들어왔을때 스토리지에 저장한 모드를 불러와서 적용

## 📦 컴포넌트 목록

### BASICS
- Button — 다양한 변형을 지원하는 버튼
- Badge — 상태, 카테고리 표시 뱃지
- Spinner — 로딩 스피너
- Skeleton — 플레이스홀더

### OVERLAY
- Modal — 버튼 클릭시 모달창 표시 
- Toast — 알림 컴포넌트
- Tooltip — 호버 정보 표시
- Notification — 알림 드롭다운

### Navigation
- Tab - 탭메뉴 클릭시 내용표시
- Breadcrumb - 네비게이션
- Pagination - 리스트 페이지네이션 
- Sidebar Nav -사이드바 nav
- Stepper 회원가입 스탭

### Data Display
- Accordion - 호버시 내용표시
- Carousel - 캐러셀
- Progress Bar - 프로그레스 바
- Card / Image Zoom

### FORM
- Form Input - 로그인 폼검사
- Toggle Switch - css로 토글스위치
- Dropdown - select
- Search Autocomplete - 검색했을때 배열에 저장된 그검색한 특정단어 인식 
- Chip Filter 
- Star Rating - 별점
- File Upload - 드롭다운 파일 저장, 클릭해서 파일저장

### Interactive
- Counter - 카운터
- Countdown Timer - 타이머 설정
- Clipboard Copy - 클립보드 복사
- Scroll Progress - 스크롤 프레그레스
